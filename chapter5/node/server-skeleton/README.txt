A barebone Node.js project for the server skeleton
==============================

Folder structure
----------------------------

- `index.js `: the server
- `api/swagger.json`: the Swagger description of your API
- `controllers/*.js `: the route handlers
- `controllers/*Service.js` : stub data access layer

Usage instructions
----------------------------

- Un-zip the downloaded file
- install the dependencies with npm:
```
npm install
```
- serve it with node:
```
node index.js
```
- it runs by default on: `http://localhost:8080` (configured in `index.js`)
